# Student Performance Analysis Dashboard

## Project Overview

This project analyzes student performance using Power BI and Excel. The dataset contains student academic and behavioral information, including study hours, attendance, sleep hours, internet usage, assignments completed, previous scores, exam scores, academic year, semester, and placement status.

The goal of this project is to understand the key factors that influence student exam performance and placement outcomes. The analysis focuses on identifying performance trends, comparing academic behavior, and presenting insights through an interactive Power BI dashboard.

## Dataset

The dataset includes two main tables:

### Student Dataset

This table includes student behavior and activity-related information such as:

* Student ID
* Study Hours
* Attendance
* Sleep Hours
* Internet Usage
* Assignments Completed
* Placement Status
* Exam Timestamp

### Student Exam Data

This table includes exam and academic time-related information such as:

* Student ID
* Exam Timestamp
* Year
* Month
* Academic Year
* Semester
* Previous Score
* Exam Score

## Dashboard Features

The Power BI dashboard includes analysis around:

* Average exam score
* Average attendance
* Student placement status
* Exam score trends by academic year and semester
* Attendance vs exam score analysis
* Study hours vs exam score analysis
* Previous score vs current exam score comparison
* Assignment completion and student performance
* Student behavior indicators such as sleep hours and internet usage

## Tools Used

* Power BI
* Microsoft Excel
* Power Query
* Data Modeling
* DAX Measures
* Data Cleaning
* Dashboard Design
* Data Visualization

## Key Insights

* Students with stronger attendance generally show better academic performance.
* Study hours and assignment completion are important indicators when reviewing exam performance.
* Previous scores can help compare student progress and identify improvement patterns.
* Placement status can be analyzed alongside exam scores and attendance to understand student readiness.
* Academic year and semester analysis helps track performance trends over time.

## Conclusion

This project demonstrates how Power BI can be used to transform student performance data into a clear and interactive dashboard. The analysis helps identify academic trends, performance gaps, and behavioral factors that may affect exam results and placement outcomes.

The project highlights skills in data cleaning, data modeling, Power BI dashboard development, KPI tracking, and business-style reporting.
